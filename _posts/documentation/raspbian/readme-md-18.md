# Raspbian

Raspbian是树莓派社区首推的操作系统。

Raspbian是基于Debian的免费操作系统，特别为树莓派度身订造。系统自带超过35,000个软件包，软件源还有海量预编译的软件，敲几个命令就可以轻松安装。

开源社区一直坚持Raspbian的开发，致力于系统的稳定性和兼容性。
